# Maintenance Ticket Skill

## Description
This skill enables Claude to assist with creating and managing maintenance tickets using the AI Maintenance Ticket Assistant application.

## Usage
When invoked, this skill helps users:
- Draft maintenance ticket descriptions
- Categorize issues appropriately
- Assess priority levels based on severity
- Suggest troubleshooting steps

## Commands
- `create-ticket` - Help draft a new maintenance ticket
- `categorize` - Suggest appropriate category for an issue
- `prioritize` - Help assess priority level
- `troubleshoot` - Provide basic troubleshooting guidance

## Context
This skill is designed for the AI Maintenance Ticket Assistant web application that manages maintenance requests for buildings, equipment, and facilities.
